# Event calendar
> Application that allows to view and manage cultural events.
> https://evcal.herokuapp.com

## General info
This version includes among others:
* An admin site for managing database.
* User authentication.
* Viewing and managing cultural events.
* Creating new events.

## Technologies
* python 3.8.0
	* Django 3.0.4
* Bootstrap 4
